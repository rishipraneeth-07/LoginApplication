# Spring Boot Login Application (Thymeleaf + Spring Security)

This is a beginner-friendly Spring Boot project that demonstrates how to implement a login page using Thymeleaf and Spring Security and
focuses on understanding authentication flow, session management, and basic security using HTML login page.

## Features

Custom login page using Thymeleaf

Authentication using Spring Security

Session-based login

Protected home page

Using Lombok 

In-memory / H2 database support (can be extended)
